# <img width="32" height="32" alt="image" src="https://github.com/user-attachments/assets/762c7f86-c49f-407c-bae1-26cbdacd7a9d" /> Installation
For the mod to work correctly, install BepInEx 4+ and PluginConfigurator/Configgy. <br>
### **Installation via r2modman (ONLY STEAM VERSION)**
Follow these steps:
- Find this mod on r2modman or just click here (WIP)
- Press install, you dont need to download other dependicies of the mod, they install automaticly
- Press "Start Modded" and enjoy...
### **How to manually install mod?**
Follow these steps:
- Download last version of the mod in Releases
- Open *.zip* file and extract folder in the ``` ULTRAKILL parent folder\ULTRAKILL\BepInEx\Plugins\ ```
- Run the game and enjoy...
